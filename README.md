# UNIT FIT1033 - 3D Village Project

## Overview
This is a 3D environment project created for **FIT1033** at Monash University. It features a stylized Japanese-inspired village, including a Torii gate, traditional housing, and environmental assets.

## Tools Used
* **Maya 2017/2027**: Modeling and UV Mapping.
* **Unity 6 (6000.2.2f1)**: Level design, lighting, and materials.
* **C# / Unity Engine**: For the interactive walkthrough.

## How to Run
1. Clone this repository.
2. Open **Unity Hub**.
3. Add this project folder.
4. Unity will rebuild the `Library` folder upon first launch (this may take a few minutes).
5. Open `Assets/Scenes/SampleScene.unity` and press **Play**.

## Project History
Originally created 7 years ago (2019) and updated in 2026 to run on Unity 6 and Apple Silicon (Metal).
